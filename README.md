# CineAI Lab Website

A short one-page website for presenting CineAI Lab to the OpenAI Startup Community.

## Files

- `index.html` — main page
- `styles.css` — design and responsive layout
- `script.js` — small footer year script
- `public/favicon.svg` — browser icon
- `public/og-image.svg` — preview image for sharing
- `vercel.json` — simple Vercel configuration

## How to publish with GitHub and Vercel

1. Create a new GitHub repository, for example: `cineai-lab`.
2. Upload all files from this folder to the repository.
3. Go to Vercel.
4. Choose **Add New Project**.
5. Import the GitHub repository.
6. Keep the default settings. No build command is needed.
7. Deploy.

## Before publishing

In `index.html`, replace:

```html
hello@cineailab.example
```

with your real contact email, or remove the contact button if you prefer.

## Notes

This is a static website. It does not need Node.js, a database, or a backend.
